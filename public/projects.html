Google Cybersecurity Cert — Security Exercises

Skills: tcpdump, NIST CSF, Incident Response, Compliance Auditing

A set of hands-on security exercises completed as part of the Google Cybersecurity Professional Certificate:

  • DoS SYN Flood Analysis — Identified a SYN flood attack from packet logs, traced TCP handshake exhaustion to a 504 gateway timeout.
  • Network Traffic Analysis — Used tcpdump to diagnose DNS port 53 unreachable errors, identified root cause and proposed firewall/DNS remediation.
  • DDoS Incident Report (NIST CSF) — Full post-mortem of an ICMP flood attack mapped across all five NIST CSF functions: Identify, Protect, Detect, Respond, Recover.
  • Compliance Audit — GDPR and PCI DSS gap analysis for a fictional company, flagging missing least privilege controls and unencrypted payment data.
