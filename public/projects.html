Wheatly

Tech: Python, Ollama, LM Studio, local LLM

Before I had access to tools like Claude Code, I built my own. Wheatly is a local AI assistant integrated directly with my Obsidian vault — handling daily note creation, habit tracking, task rollover, inactivity checks, weekly summaries, and a persistent memory system. It has a terminal UI and a chat interface, and started as a small automation script before growing into something I actually relied on daily.

A big part of this project was learning the hardware side of local LLM inference. My machine has an AMD RX 6600 (8GB VRAM), and getting that to work with local models turned into a deep dive into GPU compute. CUDA gets all the attention but it's NVIDIA-only — on AMD you're looking at ROCm, which has much patchier support. Most tools either didn't support it or required workarounds. LM Studio ended up being the only frontend that had solid ROCm support at the time, which is how I landed on it. I learned a lot about shader architecture, VRAM constraints, and why an 8GB card hits a wall fast with larger models.

Eventually I discovered that my hardware just wasn't going to cut it for the kind of inference I wanted — and moved to Claude Code. But the project taught me more about Python architecture, GPU compute, and local AI tooling than anything else I've built.

Google Cybersecurity Cert — Security Exercises

Skills: tcpdump, NIST CSF, Incident Response, Compliance Auditing

A set of hands-on security exercises completed as part of the Google Cybersecurity Professional Certificate:

  • DoS SYN Flood Analysis — Identified a SYN flood attack from packet logs, traced TCP handshake exhaustion to a 504 gateway timeout.
  • Network Traffic Analysis — Used tcpdump to diagnose DNS port 53 unreachable errors, identified root cause and proposed firewall/DNS remediation.
  • DDoS Incident Report (NIST CSF) — Full post-mortem of an ICMP flood attack mapped across all five NIST CSF functions: Identify, Protect, Detect, Respond, Recover.
  • Compliance Audit — GDPR and PCI DSS gap analysis for a fictional company, flagging missing least privilege controls and unencrypted payment data.
